Proyectos

TeCuido

Es un proyecto profesional diseñado para una empresa de salud con diferentes servicios médicos. Está desarrollado con HTML, CSS y JavaScript, además de animaciones que complementan un impacto visual en el usuario.

TruckerFood

Es un proyecto personal el cual no contiene información verídica de una empresa, pues su objetivo está en demostrar el dominio de los lenguajes de desarrollo frontend principales (HTML, CSS y JS), además del framework Angular y la librería Bootstrap. Sumado a ello, el proyecto cuenta con desarrollo backend pues cuenta con una base de datos en la cual se pueden introducir datos que el cliente desee.
